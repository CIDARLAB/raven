Description of Assembly Method

Raven implements the 3A variant of the binary assembly method BioBricks. In Biobricks, all parts are preceded by EcoRI and XbaI restriction sites, and followed by SpeI and PstI. Digesting these restriction sites produces common sticky ends, which allow 2 different BioBrick parts to be assembled. 3A assembly differs from other variants of Biobricks in that it relies on a 3 way ligation between 2 BioBrick parts and the vector. By utilizing a 3 way ligation, BioBrick 3A assembly uses both positive and negative selection to reduce the number of undesired assembly products.

Description of Assembly Method

CPEC (Circular Polymerase Extension Cloning) is a one step multiway assembly method. Multiple double stranded DNA parts with overlapping sequences can be denatured and amplified using DNA polymermase. The final concatenated part must have homology on both ends with the assembly vector for a plasmid to form properly.

Description of Assembly Method

Gibson is a one-pot, ithothermal, multi-way assembly method. Gibson assembly relies on the combined effects of a 5' exonuclease, DNA polymerase, and a DNA ligase to assemble overlapping DNA molecules. In the first part of the reaction, overhangs between two overlapping parts are created by the exonuclease, which allows the two parts to anneal. The gaps on each strand of the annealed parts is then repaired with DNA polymerase and DNA ligase.

Description of Assembly Method

Golden Gate is a one step multi-way assembly method that relies on Type IIs restriction enzymes. Type IIs restriction enzymes cut downstream from their recognition site, which can be leveraged to generate unique overhangs. Parts with matching overhangs are first annealed and then ligated together using DNA ligase.

Description of Assembly Method

MoClo is a one step multi-way assembly method that uses common overhang junctions generated by type IIs restriction enzymes. Although the common overhang junctions are effectively assembly scars, the use of common junctions allows the reuse of parts, which reduces the cost. First parts are digested with Type IIs restriction enzymes and then the digested parts are ligated together.

Description of Assembly Method

SLIC (Sequence and Ligation Independent Cloning) is a one-pot multi-way assembly methods taht uses in vitro homologous recombination and single-strand annealing. SLIC uses exonuclease to generate single stranded DNA overhangs in inserts as well as vectors. SLIC effectively mimics in vivo homologous recombination.
